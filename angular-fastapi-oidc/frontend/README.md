# Angular Frontend

Angular frontend using angular-oauth2-oidc for OAuth2/OIDC authentication.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Configuration

Edit `src/environments/environment.ts` to configure:
- OIDC Issuer URL
- Client ID
- Redirect URI
- API URL

## Development

The app will be available at `http://localhost:4200`
